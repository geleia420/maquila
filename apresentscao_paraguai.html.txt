            <h2 class="text-3xl font-bold text-gray-800 mb-6">Regime de Maquila: Vantagem Fiscal Incomparável</h2>
            <p class="text-gray-700 mb-4 leading-relaxed">O Regime de Maquila (Lei Nº 1.064/97) é a espinha dorsal da atratividade do Paraguai para indústrias exportadoras. Ele permite:</p>
            <ul class="list-disc list-inside text-gray-700 ml-4 mb-6">
                <li>**Tributo Único de 1%:** Incide apenas sobre o valor agregado no Paraguai (valor da exportação menos custo da matéria-prima e frete). Isso substitui todos os outros impostos relacionados ao processo produtivo, resultando em uma carga tributária efetiva extremamente baixa.</li>
                <li>**Suspensão Total de Impostos de Importação:** Matérias-primas, insumos, máquinas e equipamentos são importados temporariamente sem impostos, reduzindo drasticamente os custos iniciais e operacionais.</li>
                <li>**Remessa de Lucros e Dividendos Livres de Impostos:** Garante a livre repatriação de capital e lucros sem tributação adicional, um fator crucial para investidores estrangeiros.</li>
                <li>**Isenção de IVA nas Exportações:** As exportações são isentas de IVA, otimizando ainda mais a estrutura de custos e a competitividade internacional.</li>
            </ul>
            <p class="text-gray-700 mt-4 leading-relaxed">Esta estrutura fiscal torna o Paraguai um dos locais mais competitivos para a manufatura orientada para a exportação, impactando diretamente a lucratividade e a capacidade de oferecer preços agressivos no mercado global.</p>

            <h3 class="text-2xl font-semibold text-gray-800 mt-8 mb-4">Regras de Origem MERCOSUL e Acesso Preferencial</h3>
            <p class="text-gray-700 mb-4 leading-relaxed">Sob o Regime de Maquila, as empresas podem exportar seus produtos para os países membros do MERCOSUL com tarifa zero, desde que cumpram as regras de origem estabelecidas. A regra geral exige 60% de conteúdo nacional ou 40% de conteúdo regional para mercadorias originárias de fora da extrazona. Isso é uma vantagem crítica para acessar o vasto mercado sul-americano sem barreiras tarifárias, permitindo otimizar cadeias de suprimentos globais enquanto se aproveita o acesso ao mercado regional.</p>

            <h3 class="text-2xl font-semibold text-gray-800 mt-8 mb-4">Estrutura e Suporte Governamental</h3>
            <p class="text-gray-700 mb-4 leading-relaxed">O Conselho Nacional da Indústria Maquiladora de Exportação (CNIME) é o órgão nacional de formulação de políticas e regulamentação para a indústria de Maquila, presidido pelo Ministério da Indústria e Comércio (MIC). O CNIME funciona como um ponto único de contato ("centro único de trámites") para o processamento eficiente e rápido de várias solicitações, licenças e registros relacionados a empresas maquiladoras. Essas entidades estão trabalhando ativamente para modernizar e simplificar os processos burocrática, a fim de aumentar a rastreabilidade e a transparência.</p>

            <ul class="list-disc list-inside text-gray-700 ml-4">
                <li>**Período de Experiência:** O período de experiência é de 30 dias para empregados não qualificados ou domésticos e de 60 dias para trabalhadores qualificados ou aprendizes.</li>
                <li>**Férias Anuais:** Os empregados têm direito a férias anuais remuneradas após um ano de serviço contínuo, variando de 12 dias consecutivos para menos de cinco anos de serviço, 18 dias para 5 a 10 anos, e 30 dias após dez anos de serviço contínuo.</li>
                <li>**Licença Maternidade/Paternidade:** A licença maternidade é de 18 semanas (em circunstâncias normais) ou 24 semanas (em caso de complicações médicas), com 30 dias adicionais por filho em caso de nascimentos múltiplos. Pais recém-nascidos têm direito a duas semanas de licença paternidade remunerada.</li>
                <li>**Criação de Empregos pelo Setor de Maquila:** O setor de maquiladoras é um gerador de empregos significativo e em crescimento no Paraguai, indicando um crescente pool de mão de obra industrial que adquire experiência em processos de fabricação orientados para a exportação.</li>

            <p class="text-gray-700 mb-4 leading-relaxed">O Paraguai tem demonstrado consistentemente uma forte disciplina macroeconômica, o que cria um ambiente previsível e seguro para empresas e investidores. O Produto Interno Bruto (PIB) real do país cresceu a uma taxa média anual notável de 3,7% nas últimas duas décadas (2003–2023), um desempenho que supera tanto a média da região da América Latina e Caribe (LAC) quanto a de países de renda média-alta.             <h3 class="text-2xl font-semibold text-gray-800 mt-8 mb-4">Carga Tributária Geral e Incentivos ao Investimento Estrangeiro</h3>
            <p class="text-gray-700 mb-4 leading-relaxed">O Paraguai mantém um ambiente tributário competitivo, caracterizado por uma baixa carga tributária geral, que representa 9,9% do PIB, significativamente abaixo da média de 15-20% para os países da LAC. A taxa geral de imposto de renda corporativo (Impuesto a la Renta Empresarial - IRE) é uma taxa fixa de 10% sobre o lucro líquido tributável. Regimes simplificados (SIMPLE, RESIMPLE) existem para empresas menores. Embora um imposto sobre a distribuição de dividendos (IDU) de 8% para residentes e 15% para não residentes (com variações de tratado) geralmente se aplique, o Regime de Maquila isenta explicitamente as remessas para maquiladoras, reforçando sua vantagem distinta. O Imposto ao Valor Agregado (IVA) geralmente se aplica a uma taxa de 10%, com uma taxa reduzida de 5% para certos bens e serviços essenciais. É importante ressaltar que as exportações são isentas de IVA.</p>

        // Função para mostrar uma página específica e esconder as outras
        function showPage(pageId) {
            const pages = document.querySelectorAll('.page-content');
            pages.forEach(page => {
                page.classList.remove('active');
            });

            const activePage = document.getElementById(pageId);
            if (activePage) {
                activePage.classList.add('active');
                window.scrollTo({ top: 0, behavior: 'smooth' }); // Rola para o topo da nova página
            }
        }

        // Inicializa a página principal ao carregar
        document.addEventListener('DOMContentLoaded', () => {
            showPage('home-page');
        });

        // Chatbot functionality (only for the chatbot page)
        const chatMessagesChatbot = document.getElementById('chat-messages-chatbot');
        const userInputChatbot = document.getElementById('user-input-chatbot');
        const sendButtonChatbot = document.getElementById('send-button-chatbot');
        const loadingIndicatorChatbot = document.getElementById('loading-indicator-chatbot');

        let chatHistoryChatbot = [{
            role: "model",
            parts: [{ text: "Olá! Sou seu assistente para tirar dúvidas sobre as vantagens de estabelecer uma empresa no Paraguai para fabricação de portas de vidro para refrigeradores comerciais. Como posso ajudar hoje?" }]
        }];

        // Função para adicionar uma mensagem ao chat do chatbot
        function addMessageChatbot(sender, text) {
            const messageDiv = document.createElement('div');
            messageDiv.classList.add('message-bubble');
            if (sender === 'user') {
                messageDiv.classList.add('user-message');
            } else {
                messageDiv.classList.add('ai-message');
            }
            messageDiv.innerHTML = text.replace(/\n/g, '<br>'); // Converte quebras de linha para HTML
            chatMessagesChatbot.appendChild(messageDiv);
            chatMessagesChatbot.scrollTop = chatMessagesChatbot.scrollHeight; // Rola para o final
        }

        // Função para enviar mensagem para a API Gemini (chatbot)
        async function sendMessageToGeminiChatbot(message) {
            loadingIndicatorChatbot.classList.remove('hidden'); // Mostra o indicador de carregamento
            sendButtonChatbot.disabled = true; // Desabilita o botão de enviar

            chatHistoryChatbot.push({ role: "user", parts: [{ text: message }] });

            const payload = { contents: chatHistoryChatbot };
            const apiKey = ""; // Canvas will provide the API key at runtime.
            const apiUrl = `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=${apiKey}`;

            try {
                const response = await fetch(apiUrl, {
                    method: 'POST',
                    headers: { 'Content-Type': 'application/json' },
                    body: JSON.stringify(payload)
                });

                const result = await response.json();

                if (result.candidates && result.candidates.length > 0 &&
                    result.candidates[0].content && result.candidates[0].content.parts &&
                    result.candidates[0].content.parts.length > 0) {
                    const aiResponse = result.candidates[0].content.parts[0].text;
                    addMessageChatbot('ai', aiResponse);
                    chatHistoryChatbot.push({ role: "model", parts: [{ text: aiResponse }] });
                } else {
                    addMessageChatbot('ai', 'Desculpe, não consegui gerar uma resposta. Por favor, tente novamente.');
                }
            } catch (error) {
                console.error('Erro ao chamar a API Gemini:', error);
                addMessageChatbot('ai', 'Ocorreu um erro ao se comunicar com a IA. Por favor, tente novamente mais tarde.');
            } finally {
                loadingIndicatorChatbot.classList.add('hidden'); // Esconde o indicador de carregamento
                sendButtonChatbot.disabled = false; // Habilita o botão de enviar
                userInputChatbot.focus(); // Coloca o foco de volta no campo de entrada
            }
        }

        // Evento de clique no botão de enviar do chatbot
        sendButtonChatbot.addEventListener('click', () => {
            const message = userInputChatbot.value.trim();
            if (message) {
                addMessageChatbot('user', message);
                userInputChatbot.value = ''; // Limpa o campo de entrada
                sendMessageToGeminiChatbot(message);
            }
        });

        // Evento de tecla "Enter" no campo de entrada do chatbot
        userInputChatbot.addEventListener('keypress', (e) => {
            if (e.key === 'Enter') {
                sendButtonChatbot.click(); // Simula o clique no botão de enviar
            }
        });

        // Foca no campo de entrada do chatbot ao mostrar a página do chatbot
        document.getElementById('chatbot-page').addEventListener('transitionend', (event) => {
            if (event.propertyName === 'display' && document.getElementById('chatbot-page').classList.contains('active')) {
                userInputChatbot.focus();
            }
        });
